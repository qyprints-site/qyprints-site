# QYPrints Industrial Site

Professional B2B website for ShenZhen QiYao Tec Co., Ltd. (QYPrints), specializing in industrial label printing and custom packaging.

## Features
- **Modern Industrial Design**: High-end aesthetic inspired by industry leaders.
- **Product Solutions**: Specialized pages for Labels and Custom Boxes.
- **Compliance Focused**: Dedicated sections for ISO, RoHS, FSC, and REACH certifications.
- **B2B Inquiry System**: Integrated WhatsApp floating button and professional contact forms.
- **SEO Optimized**: Advanced metadata and JSON-LD structured data for Google rankings.

## Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`
